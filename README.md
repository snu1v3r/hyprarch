# Hypr-Arch
This is a repo to prepare a Archlinux install using Hyprland.


## Installation
For installation you can use the following line:

```bash
eval "$(wget -qO- https://raw.githubusercontent.com/snu1v3r/hyprarch/main/install.sh)"
```
